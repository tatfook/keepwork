## @mixLayer

## 参数说明

```
- styleID : //当前样式id

#media
- src  : //当前图片地址
- link : //当前图片跳转链接

#title
- name : //当前标题内容
- link : //当前标题跳转链接

#subtitle
- name : //当前副标题内容
- link : //当前副标题跳转链接

#paragraph
- data : //当前文档内容

#button  
- name :  //按钮名称
- link :  //按钮点击跳转链接
- target :  //链接从新窗口打开或本窗口打开

- src  : //当前图片地址
```
